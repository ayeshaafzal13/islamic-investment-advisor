# 🕌 Islamic Investment Advisor

An **expert system** that helps Muslims determine if an investment is **Halal (permissible)** or **Haram (forbidden)** according to Islamic finance principles, while also checking suitability for beginners.

## 🎯 What This Project Does

This is a **rule-based expert system** (not AI/ML) that makes investment recommendations using logical rules from Islamic finance. It answers three questions for any investment:

1. **Is it Halal or Haram?** (Based on Riba, Gharar, business type, and asset-backing)
2. **Is it suitable for a beginner?** (Based on capital, risk comfort, investment horizon)
3. **What is the risk/return profile?** (Low/Moderate/High + time horizon)

## 🚀 How to Run

### Prerequisites
- Python 3.8 or higher installed on your computer
- No external libraries required (uses only Python standard library)

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/ayeshaafzal13/islamic-investment-advisor.git

# 2. Navigate to project folder
cd islamic-investment-advisor

# 3. Run the application
python main.py
